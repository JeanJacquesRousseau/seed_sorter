Seed sorter Project
===============

To compile this project, you need openCV-3.2 for raspberry pi 3
1. Update your system
	```bash 
	sudo apt-get update
	sudo apt-get upgrade
	sudo rpi-update
	sudo reboot
	```
2. Install developpers tools
	```bash
	sudo apt-get install build-essential cmake cmake-curses-gui pkg-config
	```
